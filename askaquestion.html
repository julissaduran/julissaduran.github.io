Ask a Question


For this assignment, I decided to re-create a Magic 8 Ball using p5.js. The Magic 8 Ball is a classic toy that most of us have played with. If you have never played with this toy before, you ask it a question and shake it for an answer. In this case, pressing any of the keys from 1 to 0 on your keyboard will give you a different response. Make sure you ask a question before to get your fortune!

function clear_print(){
//this will do one of two things,x clears the screen by resetting
the background
// p calls the routine saveme, which saves a copy of the screen
if(key =='x'||key == 'X'){

} else if(key == 'p'||key =='P'){
saveme();// call saveme which saves an image of the screen
}
}
function saveme(){
//this will save the name as the initials,date,time,and a millis
counting number.
//it will always be larger in value then the last one.
filename=initials+day() + hour() + minute() +second();

just took one

}

more than one per second
